# AndroidLibrary
[![](https://jitpack.io/v/zwna/AndroidLibrary.svg)](https://jitpack.io/#zwna/AndroidLibrary)
使用方法：
    1.在项目的根目录下的build.gradle的添加
    allprojects {
    		repositories {
    			maven { url 'https://jitpack.io' }
    		}
    	}
    2.添加依赖
    implementation 'com.github.zwna:AndroidLibrary:1.1.2'
